The underlying workings of java libraries is that each library is a class with objects where these objects can be accessed when the class is stored inside a variable.
Eg: in the IO program, im using buffered writer class/library, to access the objects and methods within that class i create a variable called writer with the data type of Buffered writerand I store a new buffered writer in that variable.

The same way when taking input, the scanner is stored within a userdefined variable of the scanner class type.

If the program for creating a file and writing into it occurs more than once, the file will be overwritten.
